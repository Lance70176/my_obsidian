# 歡迎使用 my_obsidian 👋

這是一個本地 Markdown 知識庫 App:**編輯、預覽、雙向連結、匯出 HTML**。

## 快速開始

- 左側是檔案樹,點擊筆記即可開啟
- 上方切換 **編輯 / 分割 / 預覽** 三種模式(⌘1 / ⌘2 / ⌘3)
- 內容會自動儲存,也可以按 ⌘S 立即儲存
- 按 ⌘N 建立新筆記

## 雙向連結

用 `[[筆記名稱]]` 連到其他筆記,例如:[[功能介紹]]、[[教學/匯出說明]]。

連到不存在的筆記時會顯示為虛線,點擊即可建立:[[我的新想法]]。
