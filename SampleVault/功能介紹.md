## Markdown 語法支援

支援 GFM(GitHub Flavored Markdown):表格、任務清單、刪除線、自動連結等。

| 功能 | 語法 | 支援 |
| ---- | ---- | :--: |
| 粗體 | `**text**` | ✅ |
| 雙向連結 | `[[筆記名稱]]` | ✅ |
| 程式碼高亮 | ` ```js ` | ✅ |
| 表格 | `\| a \| b \|` | ✅ |

## 程式碼高亮

```js
// highlight.js 自動上色
function greet(name) {
  const message = `Hello, ${name}!`;
  console.log(message);
  return message;
}
greet('MyObsidian');
```

```python
def fib(n: int) -> int:
    a, b = 0, 1
    for _ in range(n):
        a, b = b, a + b
    return a
```
